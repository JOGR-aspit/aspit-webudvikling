# W1 Exercises

This folder contains practice exercises for W1 curriculum topics.

## Exercise Structure

Each curriculum topic has **3 exercises** with increasing difficulty:

- **Level A (Easiest)**: Full boilerplate with detailed instructions
- **Level B (Moderate)**: Partial boilerplate with moderate guidance
- **Level C (Challenging)**: Minimal starter with open-ended challenges

## Progression

Complete exercises in order: **Level A → Level B → Level C**

This gradual progression helps you build skills and confidence step by step.

## Exercise Topics

### Exercise 1: Text in HTML

**Folder**: `exercise-01-text/`

**Learning Objectives**:
- Practice heading tags (h1-h6) with proper hierarchy
- Create paragraphs with `<p>` tags
- Use `<strong>` and `<em>` tags for semantic text formatting
- Add HTML comments to document code

**Difficulty Levels**:

#### Exercise 1a: Full Guidance

**File**: `exercise-01a-full.html`

**Tasks**:
1. Create main heading (h1): "My Favorite Hobbies"
2. Create subheading (h2): "Hobbies I Enjoy"
3. Write introduction paragraph about hobbies in general
4. Create subheading (h2): "My Favorite Hobby"
5. Write about favorite hobby with `<strong>` and `<em>` tags
6. List 3+ other hobbies in separate paragraphs
7. Add HTML comment at the top explaining the page

**Estimated Time**: 20-30 minutes

#### Exercise 1b: Moderate Guidance

**File**: `exercise-01b-moderate.html`

**Tasks**:
1. Create main heading (h1) for the page
2. Create subheading (h2) for sections
3. Write introduction paragraph with `<p>` tag
4. Create content about favorite hobby with emphasis tags
5. List 3 other hobbies in paragraphs
6. Add HTML comment explaining structure

**Estimated Time**: 25-35 minutes

#### Exercise 1c: Minimal Guidance

**File**: `exercise-01c-minimal.html`

**Tasks**:
Create a webpage about your favorite hobbies with:
1. Proper heading hierarchy (h1, h2)
2. Multiple paragraphs
3. `<strong>` and `<em>` tags for emphasis
4. HTML comments explaining structure

**Estimated Time**: 35-45 minutes

---

### Exercise 2: Lists in HTML

**Folder**: `exercise-02-lists/`

**Learning Objectives**:
- Create unordered lists (`<ul>`) for non-sequential items
- Create ordered lists (`<ol>`) for sequential items
- Practice nested lists (lists within lists)
- Use semantic list structures

**Difficulty Levels**:

#### Exercise 2a: Full Guidance

**File**: `exercise-02a-full.html`

**Tasks**:
1. Create morning routine section with ordered list (5 items)
2. Create food list section with unordered list (4-6 items)
3. Create shopping list section with nested lists (3 categories, 2-3 items each)
4. Add HTML comment explaining page structure

**Estimated Time**: 25-35 minutes

#### Exercise 2b: Moderate Guidance

**File**: `exercise-02b-moderate.html`

**Tasks**:
1. Create ordered list for morning routine (5 activities)
2. Create unordered list for food items (4-6 items)
3. Create nested list for shopping (3 categories with sub-items)
4. Add HTML comment

**Estimated Time**: 30-40 minutes

#### Exercise 2c: Minimal Guidance

**File**: `exercise-02c-minimal.html`

**Tasks**:
Create a webpage about your daily routine with:
1. Ordered list (sequential items)
2. Unordered list (non-sequential items)
3. At least one nested list
4. Proper list tags (`<ul>`, `<ol>`, `<li>`)
5. HTML comments

**Estimated Time**: 40-50 minutes

---

### Exercise 3: Links and Images

**Folder**: `exercise-03-links-images/`

**Learning Objectives**:
- Create links with `<a>` tags and `href` attributes
- Add images with `<img>` tags and proper alt text
- Use relative file paths
- Make images clickable (image links)
- Resize images appropriately

**Difficulty Levels**:

#### Exercise 3a: Full Guidance

**File**: `exercise-03a-full.html`

**Tasks**:
1. Create main heading: "My Photo Gallery"
2. Add introduction paragraph
3. Add photo from `images/photo1.jpg` with alt text (width 300)
4. Add caption paragraph
5. Create external link with `target="_blank"`
6. Add second photo that's clickable (wrapped in `<a>`)
7. Add email link (`mailto:`)
8. Add HTML comment

**Preparation**: Place photos in `images/` folder as `photo1.jpg` and `photo2.jpg`

**Estimated Time**: 30-40 minutes

#### Exercise 3b: Moderate Guidance

**File**: `exercise-03b-moderate.html`

**Tasks**:
1. Create photo gallery heading
2. Add introduction paragraph
3. Add image with src, alt, width attributes
4. Create external link opening in new tab
5. Make an image clickable (nested in `<a>`)
6. Add email link (`mailto:`)
7. Add HTML comment

**Estimated Time**: 35-45 minutes

#### Exercise 3c: Minimal Guidance

**File**: `exercise-03c-minimal.html`

**Tasks**:
Create a photo gallery webpage with:
1. At least 2 images with proper alt text
2. External link opening in new tab
3. At least one clickable image
4. Email link (`mailto:`)
5. Appropriately sized images
6. Relative file paths for local images
7. HTML comments

**Preparation**: Place photos in `images/` folder

**Estimated Time**: 45-60 minutes

## How to Use These Exercises

1. **Start with Level A** (Easiest) for each topic
2. **Follow embedded instructions** in each exercise file
3. **Open in browser frequently** to see your work as you go
4. **Complete Level B** (Moderate) to build skills
5. **Complete Level C** (Challenging) to demonstrate mastery
6. **Reference curriculum** if you need help: Lessons 03, 04, 05

## Tips for Success

- **Read instructions carefully** - Each task is numbered and specific
- **Open in browser frequently** - See your work immediately
- **Use hints** (in Level B) if you get stuck
- **Check against curriculum** - Review corresponding lesson if unsure
- **Take your time** - Learning isn't a race!

## Success Criteria

You've mastered these exercises when you can:
- [ ] Complete all 3 difficulty levels for Exercise 1 (Text)
- [ ] Complete all 3 difficulty levels for Exercise 2 (Lists)
- [ ] Complete all 3 difficulty levels for Exercise 3 (Links & Images)
- [ ] Create proper heading hierarchies without thinking
- [ ] Use semantic tags (`<strong>`/`<em>` instead of `<b>`/`<i>`)
- [ ] Create unordered and ordered lists from memory
- [ ] Build nested lists with proper structure
- [ ] Add images with descriptive alt text
- [ ] Create links with proper attributes
