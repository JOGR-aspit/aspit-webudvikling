# Images Folder

Add your game images to this folder.

**You need at least 3 images** for the gallery section:
- Screenshots from the game
- Official artwork
- Character renders
- Concept art

**Naming tips:**
- Use lowercase letters
- Use hyphens instead of spaces: `screenshot-1.jpg`, `character-art.png`
- Include the file extension: `.jpg`, `.png`, or `.webp`

**Where to find images:**
- Official game website
- Press kits (often linked from official sites)
- Game Steam page screenshots
- Fan sites (make sure you have permission or use official assets)
