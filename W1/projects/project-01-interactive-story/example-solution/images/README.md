# Images for "The Missing Heirloom" Example Solution

This folder should contain images for the example story. For the example to display
properly, you need to add the following images:

## Required Images

1. **mansion.jpg** - A grand Victorian mansion on a hill
   - Suggested search: "Victorian mansion exterior" on Unsplash or Pexels

2. **library.jpg** - A grand library with bookshelves and fireplace
   - Suggested search: "old library books fireplace" on Unsplash or Pexels

3. **kitchen.jpg** - A warm Victorian-style kitchen
   - Suggested search: "Victorian kitchen vintage" on Unsplash or Pexels

4. **garden.jpg** - An English garden with paths and shed
   - Suggested search: "English garden path" on Unsplash or Pexels

5. **treasure-found.jpg** - A dusty room/shelf with treasures
   - Suggested search: "antique shelf treasures" or "dusty attic" on Unsplash or Pexels

6. **disappointed.jpg** - A disappointed person or detective scene
   - Suggested search: "disappointed businessman" or "detective office" on Unsplash or Pexels

## Free Image Sources

- **Unsplash**: https://unsplash.com (Free high-quality photos)
- **Pexels**: https://pexels.com (Free stock photos)
- **Pixabay**: https://pixabay.com (Free images)

## Image Tips

- Choose images that match the mood of the story
- Keep images reasonably sized (400-600px width works well)
- Save as JPG format for photos
- Always write descriptive alt text in your HTML

## Note for Students

Students creating their own stories should find their own images from these
free sources. Remind them to always include descriptive alt text!
